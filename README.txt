Instale Node.js. Lo necesitaremos para cargar, administrar dependencias y ejecutar nuestra herramienta de compilación.
https://nodejs.org/en

Instale three.js y una herramienta de compilación, Vite, usando una terminal en la carpeta de su proyecto.
--Escribir en terminal de Visual Studio Code:
  # three.js
  npm install --save three
  
  # vite
  npm install --save-dev vite

Una vez descargados abrir un host local como :
"npx vite" y despues presionar "o"
